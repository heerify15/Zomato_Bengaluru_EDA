# 🍽️ Zomato Bengaluru Restaurants & Trends Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Zomato Bengaluru restaurant dataset to uncover insights about restaurant ratings, customer preferences, online ordering trends, table booking behavior, restaurant types, service categories, costs, locations, and popular restaurant chains.

The objective is to transform raw restaurant data into meaningful business insights through data cleaning, preprocessing, visualization, and statistical analysis.

---

## 🎯 Objectives

* Understand the structure and quality of the dataset.
* Clean and preprocess restaurant data.
* Analyze customer preferences and restaurant trends.
* Identify top-performing locations and restaurant categories.
* Explore the relationship between ratings, services, costs, and restaurant popularity.
* Visualize key insights using Python data visualization libraries.

---

## 📂 Dataset

**Dataset:** Zomato Bengaluru Restaurants Dataset
**Link:** https://www.kaggle.com/datasets/rajeshrampure/zomato-dataset/data

The dataset contains information such as:

* Restaurant Name
* Location
* Rating
* Votes
* Approximate Cost for Two People
* Online Order Availability
* Table Booking Availability
* Restaurant Type
* Service Type
* Cuisines

The following data cleaning steps were performed:

* Removed redundant columns
* Renamed columns for better readability
* Removed duplicate records

* Fixed inconsistent values

## 📊 Exploratory Data Analysis

### 1. Online Ordering Analysis

* Restaurants delivering online vs offline
* Distribution of online ordering facilities

### 2. Table Booking Analysis

* Restaurants allowing table booking
* Impact of table booking on ratings

### 3. Location-Based Analysis

* Best locations based on average restaurant ratings
* Number of restaurants across locations
* Relationship between location and rating
* Most expensive dining locations

### 4. Restaurant Type Analysis

* Distribution of restaurant types
* Restaurant type vs rating
* Top-rated restaurant categories

### 5. Service Type Analysis

  * Delivery
  * Dine-out
  * Cafes
  * Buffet
  * Desserts
  * Pubs & Bars

### 6. Cost Analysis

* Cost distribution of restaurants

### 7. Restaurant Chain Analysis

---

## 📈 Key Insights

* Delivery restaurants dominate the Bengaluru food market.
* Restaurants offering table booking generally receive higher ratings.
* Certain locations consistently host highly-rated restaurants.
* Premium dining locations tend to attract restaurants with strong customer ratings.
* Popular restaurant chains combine high outlet presence with strong customer engagement.
* Service type significantly influences customer ratings and restaurant performance.
